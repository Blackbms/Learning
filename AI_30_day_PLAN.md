# 🧠 30-Day AI Engineering Plan (Detailed)

## Structure
Each day = 15 minutes
- Learn (5 min)
- Do (5–7 min)
- Reflect (3–5 min)

---

# WEEK 1 — Prompt Fundamentals

## Day 1 — LLM Basics
**Learn:** LLM = next-token predictor  
**Do:** Ask same question 3 ways  
**Prompt:**
> Explain X in 1 sentence, bullets, and like I’m 5  
**Reflect:** How did structure change output?

## Day 2 — Prompt Structure
**Learn:** Role / Context / Task  
**Do:** Rewrite messy prompt  
**Prompt:**
> Role: expert  
> Context: X  
> Task: Y  
> Format: bullets  
**Reflect:** Was output clearer?

## Day 3 — Output Control
**Learn:** Structured outputs  
**Do:** Force JSON  
**Prompt:**
> Return JSON: {name, difficulty, value}  
**Reflect:** Did it follow format?

## Day 4 — Roles
**Learn:** Roles affect tone  
**Do:** Compare outputs  
**Prompt:**
> You are a senior engineer. Design X  
**Reflect:** What changed?

## Day 5 — Iteration
**Learn:** First output = draft  
**Do:** Improve twice  
**Prompt:**
> Improve above. Be more concrete  
**Reflect:** Better than first?

## Day 6 — Constraints
**Learn:** Limits improve quality  
**Do:** Add constraints  
**Prompt:**
> Under 100 words, 1 example  
**Reflect:** Tighter output?

## Day 7 — Mini Project
**Build:** Prompt template  
**Prompt:**
> Create reusable prompt template  
**Reflect:** Reusable?
